Política de Privacidade

Atualizado em: 13 September 2025

Este aplicativo ("Points & Coupons") é operado pela CE global. A versão atual para iOS não coleta dados pessoais, não exige cadastro e não envia seus dados para nossos servidores. Todas as informações inseridas permanecem no seu dispositivo.

Resumo

  • Nós não coletamos dados pessoais (nome, e-mail, telefone, localização precisa etc.).
  • Sem conta, sem cadastro.
  • Os dados (como códigos de barras/QR, nomes de marcas, datas de validade, notas) são armazenados somente no dispositivo.
  • Se esta política mudar no futuro (por exemplo, inclusão de anúncios ou analytics), esta página será atualizada.

Dados que não coletamos

Não coletamos nome, e-mail, número de telefone, contatos, localização precisa, informações de pagamento ou identificadores de dispositivo para rastreamento.

Dados processados no dispositivo

Para fornecer os recursos do app, alguns dados são processados localmente no seu dispositivo, incluindo:

  • Imagens/logotipos que você selecionar (opcional).
  • Códigos de barras/QR e informações relacionadas (por exemplo, nome da marca, número, data de validade, notas).
  • Lembretes locais (notificações), caso você os ative.

Esses dados não são enviados para nossos servidores.

Permissões

  • Câmera (opcional): para escanear códigos de barras/QR.
  • Fotos (opcional): para escolher imagens/logos da galeria.
  • Notificações (opcional): para lembretes de validade/notas.

Compartilhamento de dados

Não compartilhamos seus dados com terceiros.

Segurança e backup

Os dados são armazenados localmente no sandbox do iOS. Se você ativar o backup do iCloud, os dados do app podem ser incluídos no seu backup do iCloud conforme as configurações da Apple.

Crianças

Este app não é destinado a menores de 13 anos e não coletamos, conscientemente, dados de crianças.

Alterações nesta política

Podemos atualizar esta política periodicamente. A data de atualização será exibida no topo desta página.

Contato

Se você tiver dúvidas sobre esta política, entre em contato por meio do link de Suporte do app na App Store.
