Kebijakan Privasi

Pembaruan terakhir: 13 September 2025

Aplikasi ini ("Points & Coupons") dioperasikan oleh CE global. Versi iOS saat ini tidak mengumpulkan data pribadi, tidak mewajibkan pendaftaran, dan tidak mengirimkan data Anda ke server kami. Semua data yang Anda masukkan tetap disimpan di perangkat Anda.

Ringkasan

  • Kami tidak mengumpulkan data pribadi (nama, email, nomor telepon, lokasi presisi, dll.).
  • Tidak ada akun, tidak ada pendaftaran.
  • Data (seperti barcode/QR, nama merek, tanggal kadaluarsa, catatan) disimpan hanya di perangkat.
  • Jika kebijakan berubah di masa depan (misalnya penambahan iklan atau analitik), halaman ini akan diperbarui.

Data yang tidak kami kumpulkan

Kami tidak mengumpulkan nama, alamat email, nomor telepon, kontak, lokasi presisi, informasi pembayaran, atau pengenal perangkat untuk pelacakan.

Data yang diproses di perangkat

Untuk menyediakan fitur aplikasi, beberapa data diproses secara lokal di perangkat Anda, termasuk:

  • Gambar/logo yang Anda pilih sendiri (opsional).
  • Barcode/QR code dan informasi terkait (misalnya nama merek, nomor, tanggal kadaluarsa, catatan).
  • Pengingat lokal (notifikasi) jika Anda mengaktifkannya.

Data tersebut tidak dikirim ke server kami.

Izin (Permissions)

  • Kamera (opsional): untuk memindai barcode/QR.
  • Foto (opsional): untuk memilih gambar/logo dari galeri.
  • Notifikasi (opsional): untuk pengingat kadaluarsa/nota.

Berbagi data

Kami tidak membagikan data Anda kepada pihak ketiga.

Keamanan dan cadangan

Data disimpan secara lokal di perangkat dalam sandbox iOS. Jika Anda mengaktifkan pencadangan iCloud di perangkat, data aplikasi mungkin termasuk dalam cadangan iCloud milik Anda sesuai pengaturan Apple.

Anak-anak

Aplikasi ini tidak ditujukan untuk anak di bawah 13 tahun dan kami tidak dengan sadar mengumpulkan data dari anak-anak.

Perubahan kebijakan ini

Kami dapat memperbarui kebijakan ini dari waktu ke waktu. Tanggal pembaruan akan ditampilkan di bagian atas halaman.

Kontak

Jika Anda memiliki pertanyaan tentang kebijakan ini, silakan hubungi kami melalui tautan App Support pada halaman App Store.
